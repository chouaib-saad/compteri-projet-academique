Il est fortement recommandé de mettre à jour le firmware et le contenu du répertoire /html sur la carte SD en même temps !

1. Mise à jour du logiciel
(Takes about 60s)

2. Mettre à jour le répertoire "/html"

3. Reboot

4. Télécharger la définition du réseau neuronal (fichier tfl/tflite)

Le fichier doit être activé dans le fichier config.ini.
