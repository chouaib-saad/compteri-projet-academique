Paramètres de configuration généraux

Ici, vous définissez un paramètre supplémentaire. Les paramètres doivent correspondre à une configuration normale.

Ce n'est que si vous souhaitez vous connecter à un courtier MQTT que vous devez ajuster les paramètres correspondants.

N'oubliez pas de sauvegarder vos modifications avec "Update Config.ini" !.
Vous ne devez pas redémarrer ici, mais laissez le mode de configuration à la page suivante.
