Définir les chiffres

Ici, vous définissez vos chiffres que vous voulez lire. Si vous avez plus d'un numéro sur la lecture, vous pouvez définir plusieurs numéros avec le sélecteur "Numéro". Là, vous pouvez également définir de nouveaux numéros.

Avec le menu déroulant "ROI x" vous pouvez changer entre les différents chiffres. Marquez-les avec la souris ou les coordonnées.
Pour créer de nouvelles ROI, utilisez "Nouvelles ROI". L'ordre des ROI définit la position dans la lecture.
Vous pouvez le changer avec "move Next" / "move Previous".

N'oubliez pas de sauvegarder vos modifications avec "Save all to Config.ini" !.
