Références d'alignement

Deux références d'alignement opposées sont nécessaires pour identifier des points fixes uniques sur l'image.

Marquez la référence par glisser-déposer avec la souris ou avec les coordonnées et appuyez sur "Mettre à jour la référence".
Vous pouvez basculer entre les deux références avec "Sélectionner une référence".

N'oubliez pas de sauvegarder vos modifications avec "Save to Config.ini" !.
